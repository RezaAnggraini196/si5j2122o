  2. Jika koneksi gagal bukalah file wp-config.php klik kanan open with sublime/notepad, lihatlah baris ke 23, ke xampp control panel lalu klik admin. Jika sudah klik database, cek db_si5j2122o buka isi database tersebut
  3. Membahas mengenai Posting, Jendela Posting ( Block Visual dan HTML ) tambah,ubah,hapus
    1. Didalam dasbor klik pos tambah baru, jika sudah membuat judul bisa memanfaatkan bold,underline,link.
    2. Jika ingin membutuhkan blok untuk html, klik sunting sebagai HTML, setelah selesai sunting secara visual.
    3. Cara menambahkan gambar, cari digoogle lalu klik kanan pada gambar tersebut klik save image as, letakkan didalam folder. Cara mengupload gambar didalam web server , klik tanda + di kiri atas cari di percarian dengan nama gambar klik gambar lalu drag gambar tersebut. Cara manualnya klik unggah lalu terapkan gambar tersebut.
    4. Cara mengupload video, dengan cara download aplikasi Everything. Download Instaler yang 64-bit. Kegunaan dari Everything itu untuk mencari file secara cepat. Jika sudah ketemu filenya, upload ke web server tersebut video tersebut bisa diambil didalam folder, dan di link youtube.
    5. Untuk cara mengubah pos tersebut, klik dasbor lalu klik pos kemudian sunting pos yang mau diubah.
    6. Untuk menghapus pos tersebut klik tong sampah .
  4. Status postingan (draft, publish, pending review, scheduled)
    1. Tambah postingan baru, membuat judul lalu klik simpan konsep, konsep inilah yang dinamakan dengan draft, kenapa ? karena kita hanya punya ide judulnya saja. Belum tau ide selanjutnya. Jika mau melanjutkan topik tersebut lalu klik sunting.
    2. Publish, adalah menerbitkan pos jika sudah selesai
    3. Pending review(perlu ditinjau) peran dari pengguna misalnya bagian dari pelanggan,kontributor, penulis, dan editor. Jika ada pending review minimal diterbitkan oleh editor . awalnya menulis sebagai kontributor, jika sudah lumayan banyak editornya yang merevisi postingan tersebut .
    4. Scheduled, adalah mengatur agar terbit di tanggal lahir masing-masing.Dengan cara pos, tambah baru. Dengan contoh judul selamat ulang tahun, isi pos tersebut ucapan selamat ulang tahun. Jika sudah aturlah jadwal dengan mengklik pengaturan, lalu klik pos, visibilitasnya diubah ke Terlindung ke sandi (jika ingin ditujukan ke orang tertentu). Lalu klik terbitkan di tanggal sebelum ulangtahun. Terakhir klik jadwalkan.
    5. Kategori , tambahkan judul lalu klik pos, scroll ke bawah tambah kategori baru lalu terbitkan
    6. Tags/label, label itu semacam digunakan untuk pencarian
    7. Komentar, sunting pos yang sudah di terbitkan lalu klik pos, jangan diberikan centang pada tulisan izinkan komentar
